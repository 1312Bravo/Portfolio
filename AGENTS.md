# AGENTS.md

## Purpose
This repository is a Quarto portfolio website for data science projects, project stories, and analysis writeups.
Use this file for local project rules and use `.agents/portfolio-writing-guide.md` for the detailed writing and structure guide.

- Follow the current user request first.
- Preserve the established portfolio voice and page structure.
- Edit source `.qmd`, CSS, and asset files.
- Do not edit generated `_site` output directly.

## Project Guide
- Before drafting, editing, or restructuring portfolio content, read `.agents/portfolio-writing-guide.md`.
- Use that guide for homepage structure, project landing pages, detailed posts, writing voice, visuals, cards, and project-specific style.

## Verification
- Preview or render the Quarto site when making structural or visual changes.
- Check links to internal posts, GitHub repositories, images, and embedded sheets.
- Confirm images render from the correct relative paths.
- Summarize what changed and note any follow-up.
